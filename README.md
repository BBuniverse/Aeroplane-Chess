### 使用

----

1. 解压 zip 包
2. 导入 Intellij
3. 点击 check 输入参加游戏的人数 (2 - 4 人)
4. 选择是否与机器人比赛, 
   1. 选择机器人难度模式(stupid or clever)
   2. 选定 机器人数量 (1-3), 机器人分配按 蓝 绿 红 的顺序
5. 选择 manual 或者 auto 模式之后点击 roll 得到摇骰子的四种情况
   1. Auto
      1. 加减乘除的结果需满足 0 到 12
      2. 越界则显示 Invalid 并且点击任意棋子不会产生任何位置移动
   2. Manual
      1. 滑动鼠标滚轮自主选择两枚骰子的数字组合
      2. 选定之后再次点击 roll 则得到两数字之和的指定步数, 同理满足 0 到 12 的范围
6. 当前时人类进行游戏时如黄色，选择骰子的情况以后，需要点击 roll 按钮确认，然后点击任意的飞机尝试飞行。
7. 如果两个骰子之和大于等于10，则不会到下一位玩家，如黄色摇到10则会使下一轮继续是自己，如果连续三次摇的不小于10则前两次运动的飞机会飞回 hangar
8. 若下一位是机器人，则需要点击 roll 然后机器人会自己选择
9. 如果想起飞任意一架停在机场的飞机, 则两枚骰子中需要存在至少一个 6
10. 步数走完停落到与飞机同色的位置上时, 直接跳到下一个同色位置 (不在降落跑道适用)
11. 如果停落在与飞机同色的捷径起点位置上时, 直接跳到捷径终点
12. 若两个 stack 相撞, 则需根据电脑掷骰子 battle, 数字小的一方的 stack 全部返回 hangar
13. 起飞后成功降落的飞机不能再次起飞
14. 当只剩下一位 player 有还没有结束时, 结束游戏
